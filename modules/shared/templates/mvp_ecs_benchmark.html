{# ---- Methodology ---- #}

Benchmark Methodology

  • Prompt-construction benchmark — measures realistic prompt sizing from the ECS Prompt Builder.
  • Local Ollama token measurement — input/output tokens captured from a local LLM run (on a suitable machine).
  • RAG-informed simulated context — representative evidence context modelled from repository structure.
  • No live RAG/PGVector/object-storage in this demo screen.
  • Enterprise values are engineering planning extrapolations, not live measurements.
