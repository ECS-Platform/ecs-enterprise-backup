# ECS ROI Model

Value model with **Conservative (80%)**, **Expected (100%)**, **Aggressive (120%)** scenarios.
Anchored to the platform's ROI workbook values. Assumptions: cost-per-hour ₹1,000; average
loaded salary ₹20,00,000; stable ECS OPEX ₹2.2 Cr at scale (₹4.0 Cr Year-1 ramp).

Scenario factors apply to value-realization metrics only (effort/hours/savings/FTE/net benefit).
Application counts, framework counts and ECS cost are held constant per scenario definition.

---

## 1. Value drivers

| Driver | Mechanism | Metric |
|---|---|---|
| Audit effort reduction | Reuse evidence across frameworks | Hours saved |
| Evidence collection savings | Automated connectors replace manual chasing | Emails / hours saved |
| Compliance productivity | Collect-once → map-to-many | FTE equivalent |
| Executive reporting savings | Live dashboards replace manual decks | Reporting hours saved |
| Audit cycle-time reduction | Always-ready evidence | Days per cycle |
| SLA breach reduction | Automation + alerts | Breach count |

## 6. Payback

- **Expected:** payback within **Year 1** at the modeled adoption; net-positive from FY27 onward
  at scale.
- **Conservative:** payback within Year 1–2; remains strongly net-positive at scale (₹114.46 Cr
  Year-7).
- **Aggressive:** payback inside Year 1; ₹171.70 Cr Year-7 net benefit.
